# - ¿Qué es un gestor de bases de datos o DBMS?

Un Sistema Gestor de Base de datos(SGBD) o DataBase Management System(DBMS) en inglés, es un sistema que nos permitirá;

- Administración, gestion y administración de base de datos.  
- Elección y manejo deestructuras necessarias para elalmacenimiento y búsqueada de información eficientemente.

- ¿Qué diferencia hay entre una base de datos centralizada y una distribuida?

En la base de datos centralizada, la información se almacena en un único lugar o(servidor.

En la base de datos distribuidas los datos son distribuidos en variedes servadores.

- Describe lo que son las bases de datos NoSQL.

La base de datos NoSQL o no relacionales son aquellas que no requieren de una estructura de datos fjitos como tablas, no garantiza Completely el sistema ACID y escala muy bien de manière horizontal.

- ¿Qué es la minería de datos?

- Cuales son los desafíos éticos y legales derivados del procesamento de una base de datos?

Por un lado tenemos los desafíos éticos que son:

- Transparency y consentimiento informado a la hora de recopilar y utilizar los datos  
- La minimización de datos recopilados y registrar la recopilación excessiva  
-Discriminatoria de ciertoos groupos etnicos o socialeconomicos bajo al diseño cuando es createo de forma no justa ni imparcial

Por otro lado está las leyes reguladoras de estaística para proteger la privacidad y los datos del usuario. Las leyes que más destacan y se usan de referencia son:

-Reglamento General de Protección de Dataos (GDPR) en la Unión Europea

-Ley de Privacidad del Consumidor de California (CCPA)
